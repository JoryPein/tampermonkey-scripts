# Tampermonkey Scripts

个人收藏的一些 Tampermonkey 用户脚本（Userscripts），主要用于增强特定网站的体验。

这些脚本使用 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展运行，支持 Chrome、Edge、Firefox 等主流浏览器。

## 脚本列表

### translate_cn_hn.js
- **功能**：将 Hacker News (news.ycombinator.com) 网站的内容翻译成简体中文。
- **适用站点**：Hacker News
- **描述**：自动或手动将页面标题、评论等内容翻译为中文，便于中文用户阅读。

### translate_cn_reddit.js
- **功能**：将 Reddit 网站的内容翻译成简体中文。
- **适用站点**：Reddit (reddit.com)
- **描述**：针对 Reddit 的帖子、评论等进行中文翻译，提升阅读体验。

## 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展。
2. 打开仓库中的脚本文件（例如：https://raw.githubusercontent.com/JoryPein/tampermonkey-scripts/main/translate_cn_hn.js）。
3. 点击“安装”按钮，Tampermonkey 会自动弹出安装窗口。
   - 或者复制脚本内容，新建脚本并粘贴。

## 注意事项

- 脚本依赖第三方翻译服务（如 Google Translate API），可能受网络或服务限制影响。
- 使用前请确保符合相关网站的条款。
- 如有问题，欢迎提交 Issue 或 Pull Request。
